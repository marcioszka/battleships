# Battleships

Battleships is a classic strategy game where players strategically position and target each other's fleet of ships. This repository contains a Python implementation of the Battleships game.

## Game Overview

This Python-based implementation of Battleships allows you to enjoy the game on your local machine. The project provides a simple, text-based interface for players to set up their fleets, take turns, and attempt to sink their opponent's ships.

## Features

- **Interactive Gameplay**: Engage in turn-based gameplay with an easy-to-use text interface.
- **Fleet Customization**: Arrange your fleet according to the rules of the game.
- **Realistic Gameplay**: Implementing classic Battleships rules for a genuine experience.
- **Player vs. Player**: Challenge your friends to battleship duels.

## Getting Started

To start playing Battleships on your local machine, follow these steps:

1. **Clone the Repository**: `git clone https://github.com/marcioszka/battleships.git`
2. **Navigate to the Project Directory**: `cd battleships`
3. **Run the Game**: Execute the `battleships.py` script to start the game.

## Game Rules

1. Each player places their fleet on a 10x10 grid.
2. Players take turns to select coordinates to target and attempt to sink their opponent's ships.
3. The game continues until one player's fleet is entirely destroyed.

For detailed game rules and instructions, refer to the in-game prompts and the code documentation.
